[.description]
--
PHC Argon2id hash of the admin password, e.g. `$argon2id$v=19$m=19456,t=2,p=1$<salt>$<hash>`. Required when `qlawkus.admin.argon2.enabled` is true; the app refuses to boot without it. Generate one as described in the Secrets guide, then store it under the keystore alias `qlawkus.admin.password-hash` or the environment variable `QLAWKUS_ADMIN_PASSWORD_HASH`.

Turns on the database-free Argon2id admin `IdentityProvider`. Off by default so a consumer using a different realm (for example the embedded properties realm) is untouched. This is a build-time gate: it is read during augmentation to include or drop the provider bean, so changing it needs a rebuild.
