= Qlawkus

Qlawkus is a single-user, self-hosted autonomous agent built as a set of Quarkus extensions (`client/`, `tools/`, `messaging/`) wired together by a deployable app (`app/`).

It has dynamic cognition (a "soul"), a layered long-term memory, Google Workspace tools, multi-provider messaging (Discord, Telegram) with voice in and out, and a sandboxed shell.

== Quick start

[source,bash]
----
# Dev mode (auto-provisions Postgres + Ollama via Dev Services, no .env needed)
mvn install -pl client -am -DskipTests
cd app && mvn quarkus:dev          # http://localhost:8080

# Talk to it
curl -X POST http://localhost:8080/api/chat/sync -u qlawkus:qlawkus \
  -H 'Content-Type: application/json' -d '{"message":"Hello, who are you?"}'
----
